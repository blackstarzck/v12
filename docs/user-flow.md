# TALKPIK AI 유저 플로우 문서

확인 기준: 2026-04-22에 배포 사이트를 Playwright MCP로 직접 탐색한 사용자 흐름입니다.

유저 플로우는 사용자가 목표를 이루기 위해 어떤 화면을 어떤 순서로 지나가는지 정리한 문서입니다.

## 공통 진입 흐름

1. 사용자는 홈 화면에 진입합니다.
2. 왼쪽 사이드바에서 주요 영역을 선택합니다.
3. 필요하면 오른쪽 아래 AI 튜터를 열어 단어 검색, 문장 교정, Q&A, 1:1 문의를 시작합니다.
4. 언어가 필요하면 `KO`, `VI`, `EN` 버튼으로 화면 언어를 바꿉니다.
5. 사용자 정보나 목표를 바꾸려면 프로필 영역을 눌러 설정 화면으로 이동합니다.

## 홈에서 학습 시작

### 듣기/읽기 문제 생성 흐름

1. 홈에서 `듣기/읽기 집중` 카드를 누릅니다.
2. AI 맞춤 문제 생성 화면으로 이동합니다.
3. 사용자는 읽기 또는 듣기 중 하나를 고릅니다.
4. TOPIK 단계와 목표 등급을 고릅니다.
5. 문제 유형을 선택합니다.
6. 생성할 문제 수를 확인합니다.
7. `AI 문제 생성 시작하기`를 누릅니다.
8. 문제 풀이 화면으로 이동합니다.
9. 사용자는 지문 또는 음성을 확인하고 보기를 선택합니다.
10. `정답 확인` 또는 다음 문제 이동으로 풀이를 이어갑니다.

### 예외 흐름

- 문제 유형을 고르지 않고 생성 버튼을 누르면 `문제 유형을 선택해주세요!` 알림이 뜹니다.

## 홈에서 쓰기 연습 시작

### 새 쓰기 문제 생성 흐름

1. 홈에서 `쓰기 집중 연습` 카드를 누릅니다.
2. 쓰기 집중 연습 설정 화면으로 이동합니다.
3. 사용자는 51번, 52번, 53번, 54번 중 문제 유형을 선택합니다.
4. 필요하면 주제 검색 또는 주제 칩을 선택합니다.
5. 문제 유형을 선택하면 생성 버튼이 활성화됩니다.
6. 생성 버튼을 누릅니다.
7. 해당 쓰기 연습 화면으로 이동합니다.
8. 사용자는 문제를 읽고 답안을 입력합니다.
9. 추천 표현이나 필수 어휘를 참고합니다.
10. 제출하기를 눌러 답안 제출을 진행합니다.

### 확인된 예시 흐름

- 53번 유형 선택 → `환경` 주제 선택 → `53번 문제 생성하기` → 53번 쓰기 연습 화면으로 이동

## 홈에서 이어하기

1. 홈에서 `이어하기` 카드를 누릅니다.
2. 51번 실용문 빈칸 쓰기 화면으로 이동합니다.
3. 사용자는 ㉠, ㉡ 탭을 오가며 답안을 작성합니다.
4. 자동 저장 상태를 확인합니다.
5. 제출하기로 마무리합니다.

## 홈에서 약점 공략

1. 홈에서 `WEAK POINT` 카드를 누릅니다.
2. 53번 도표/그래프 쓰기 화면으로 이동합니다.
3. 사용자는 도입, 전개, 마무리 탭을 따라 답안을 구성합니다.
4. AI Writing Guide, 추천 표현, 필수 어휘를 참고합니다.
5. 원고지 미리보기를 확인합니다.
6. 제출하기로 답안을 제출합니다.

## 쓰기 피드백 확인

1. 사이드바에서 `쓰기 보관함`을 누릅니다.
2. 기본적으로 완료된 피드백 목록을 봅니다.
3. 사용자는 검색, 필터, 정렬로 원하는 피드백을 찾습니다.
4. 피드백 카드를 누릅니다.
5. 쓰기 피드백 상세 화면으로 이동합니다.
6. 사용자는 점수, 문제, AI 총평, 단계별 상세 분석을 확인합니다.
7. 필요하면 `구조 분석`, `상세 첨삭`, `연습 대화 보기`를 확인합니다.
8. 다시 연습하고 싶으면 `다시 풀기`를 누릅니다.
9. 보관이 필요하면 `PDF 저장`을 누릅니다.

### 임시 저장 답안 흐름

1. 쓰기 보관함에서 `임시 보관 내역` 탭을 누릅니다.
2. 작성 중인 답안 목록을 확인합니다.
3. 목록에는 작성 중 상태, 자동 저장 여부, 작성 글자 수가 표시됩니다.
4. 사용자는 이어서 작성할 답안을 선택합니다.

## 단어 복습

1. 사이드바에서 `단어장`을 누릅니다.
2. 카드 형태로 단어를 확인합니다.
3. 뜻과 예문을 확인합니다.
4. 이전/다음 버튼으로 다른 단어로 이동합니다.
5. 표 형태가 필요하면 `리스트 보기`를 누릅니다.
6. 외운 단어는 `암기 완료! 목록에서 제거`를 누릅니다.
7. 확인 창에서 제거 여부를 결정합니다.

## 내 서재 복습

1. 사이드바에서 `내 서재`를 누릅니다.
2. TOPIK I 또는 TOPIK II를 고릅니다.
3. 빠른 복습이 필요하면 `취약 문항 공략` 또는 `전체 무작위 복습`을 선택합니다.
4. 직접 관리하려면 `내 북마크 직접 관리하기` 영역에서 문제 목록을 봅니다.
5. 영역, 유형, 태그, 정렬 조건을 조정합니다.
6. 문제를 하나씩 풀거나 문제 바구니에 담습니다.
7. 문제 바구니에서 `풀기 시작`으로 묶음 복습을 시작합니다.

### 확인 필요 흐름

- 화면은 드래그해서 문제 바구니에 담는 구조를 안내합니다. 실제 드래그 완료 후 풀이 시작까지는 추가 검증이 필요합니다.

## 모의고사 확인과 응시

### 결과 확인 흐름

1. 사이드바에서 `모의고사 결과`를 누릅니다.
2. 최근 등급, 평균 점수, 영역별 점수, 총 응시 횟수를 확인합니다.
3. 성적 변화 추이에서 기간 또는 영역 버튼을 누릅니다.
4. 응시 기록에서 성적표를 확인합니다.
5. 더 많은 기록은 `전체보기`를 누릅니다.

### 새 모의고사 응시 흐름

1. 모의고사 결과 화면에서 `새 모의고사 응시`를 누릅니다.
2. 실전 모의고사 생성 화면으로 이동합니다.
3. 회차별 실전 모의고사 또는 영역별 집중 연습을 선택합니다.
4. TOPIK 단계를 선택합니다.
5. 시험 구성과 예상 시간을 확인합니다.
6. `시험 시작하기`를 누릅니다.
7. 실전 모의고사 풀이 화면으로 이동합니다.
8. 남은 시간을 보며 문제를 풉니다.
9. 필요한 경우 OMR 답안지를 열어 답안 현황을 확인합니다.
10. 마지막에 시험을 종료합니다.

### 시험 중 흐름

- 듣기 영역에서는 오디오 재생 영역을 사용합니다.
- `이전`, `다음`으로 문제를 이동합니다.
- `답안지 (OMR)`로 번호별 답안 상태를 확인합니다.
- `시험 종료`로 시험을 끝냅니다.

### 주의 흐름

- AI 튜터 패널이 열려 있으면 일부 시험 버튼을 가릴 수 있습니다. 이때 AI 튜터를 닫고 시험 버튼을 눌러야 합니다.

## 게시판 확인

### 공지사항 흐름

1. 사이드바 또는 홈의 소식 영역에서 `게시판`으로 이동합니다.
2. 기본 탭에서 공지사항 목록을 봅니다.
3. 전체, 중요, 안내, 학습, 점검, 당첨 분류로 목록을 좁힙니다.
4. 제목 검색으로 게시글을 찾습니다.
5. 제목을 클릭합니다.
6. 공지 상세 화면으로 이동합니다.
7. 본문을 읽고 `이전글`, `목록으로`, `다음글`을 사용합니다.

### 이벤트 흐름

1. 게시판에서 `이벤트` 탭을 누릅니다.
2. 진행중 또는 종료 이벤트 카드를 확인합니다.
3. 이벤트 제목, 설명, 기간, 담당 부서를 확인합니다.
4. 페이지네이션으로 다른 이벤트 목록을 탐색합니다.

## 프로필과 설정 관리

1. 사이드바 하단의 사용자 프로필 영역을 누릅니다.
2. 설정 화면으로 이동합니다.
3. 기본 프로필 탭에서 이름, 닉네임, 언어 설정을 확인하거나 바꿉니다.
4. 학습 목표 설정 탭에서 목표 급수와 시험일을 관리합니다.
5. 나의 배지 탭에서 획득 배지와 카테고리를 확인합니다.
6. 멤버십 및 결제 탭에서 현재 플랜과 결제 내역을 확인합니다.
7. 계정 보안 탭에서 비밀번호, 마케팅 수신 동의, 회원 탈퇴를 관리합니다.

## AI 튜터 사용

### 빠른 실행 흐름

1. 오른쪽 아래 AI 튜터 버튼을 누릅니다.
2. `단어 검색`, `문장 교정`, `Q&A`, `1:1 문의` 중 하나를 선택합니다.
3. 선택한 대화방이 열립니다.
4. 메시지를 입력합니다.
5. 전송 버튼을 누릅니다.
6. 답변을 확인합니다.

### 채팅 목록 흐름

1. AI 튜터에서 `채팅` 탭을 누릅니다.
2. 최근 대화방 목록을 봅니다.
3. 원하는 대화방을 선택합니다.
4. 대화를 이어갑니다.

### 알림 확인 흐름

1. AI 튜터에서 `알림` 탭을 누릅니다.
2. 학습 리마인더, 응원 알림, 공지사항을 확인합니다.
3. 필요한 알림을 보고 관련 화면으로 이동하거나 학습을 재개합니다.

## 언어 전환

1. 사이드바의 `KO`, `VI`, `EN` 버튼 중 하나를 누릅니다.
2. 메뉴와 주요 화면 문구가 선택한 언어로 바뀝니다.
3. 다시 `KO`를 누르면 한국어 화면으로 돌아옵니다.

## 전체적으로 잘 작동한 부분

- 홈에서 주요 학습 화면으로 이어지는 흐름은 명확합니다.
- 쓰기 생성, 쓰기 풀이, 피드백 확인의 큰 흐름은 자연스럽게 연결됩니다.
- 모의고사 결과에서 새 시험 응시까지의 흐름이 분명합니다.
- 게시판의 공지/이벤트 전환과 상세 이동이 확인됩니다.
- AI 튜터는 전역 보조 도구로 접근성이 좋습니다.

## 아직 불확실하거나 개선 검토가 필요한 부분

- 홈의 `실전 모의고사` 카드는 클릭 가능하지만 확인 중 별도 이동이 보이지 않았습니다.
- 일부 검색창은 입력 후 목록이 즉시 줄어드는 변화가 분명하지 않았습니다.
- 내 서재의 문제 바구니는 드래그 동작이 핵심인데, 완전한 드래그 후 풀이 시작 흐름은 추가 확인이 필요합니다.
- AI 튜터가 열려 있을 때 시험 화면의 OMR 버튼을 가리는 경우가 있었습니다.
- 언어 전환 후에도 일부 학습 문장과 피드백 문장은 한국어로 남았습니다.
