# 프로필 설정 IA

URL: `/profile_settings.html`

현재 React route: `/profile`

## 화면 목적

사용자 정보, 목표, 배지, 결제, 보안 설정을 관리하는 화면입니다.

## 주요 구성요소

- 설정 탭
  - 기본 프로필
  - 학습 목표 설정
  - 나의 배지
  - 멤버십 및 결제
  - 계정 보안
- 도토리 통합 회원 연결 안내

## 기본 프로필

- 프로필 이미지
- 이미지 변경
- 이름
- 닉네임
- 언어 설정
- 이메일
- 가입일
- 변경사항 저장

## 학습 목표 설정

- 목표 시험 급수
- 목표 시험일
- 현재 레벨
- 최근 접속일
- 목표 업데이트

## 나의 배지

- 배지 필터
  - 전체
  - XP 레벨
  - 학습 활동
- 배지 목록
- 페이지네이션

## 멤버십 및 결제

- 이용중인 플랜
- 다음 결제일
- 결제 내역
- 조회하기

## 계정 보안

- 현재 비밀번호
- 새 비밀번호
- 새 비밀번호 확인
- 비밀번호 변경
- 마케팅 정보 수신 동의
- 회원 탈퇴

## 사용자 조작 요소

- 탭 전환
- 이미지 변경
- 프로필 정보 수정
- 목표 업데이트
- 배지 필터링
- 결제 내역 조회
- 비밀번호 변경
- 마케팅 수신 동의 변경
- 회원 탈퇴
