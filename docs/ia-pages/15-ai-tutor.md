# AI 튜터 패널 IA

현재 React route 상태: 전용 route 없음. 앱 셸 내부 패널로 동작.

## 화면 목적

현재 페이지 위에서 바로 질문하거나 단어 검색, 문장 교정, 문의를 시작하는 보조 패널입니다.

## 주요 구성요소

- 플로팅 버튼
- 상단 제목
- 닫기 버튼
- 빠른 실행 버튼
  - 단어 검색
  - 문장 교정
  - Q&A
  - 1:1 문의
- 최근 대화방
- 하단 탭
  - 홈
  - 채팅
  - 알림
- 메시지 입력창
- 전송 버튼

## 사용자 조작 요소

- 플로팅 버튼으로 열기
- 닫기
- 빠른 실행 선택
- 최근 대화방 열기
- 채팅 내역 보기
- 알림 보기
- 메시지 입력 후 전송

## 확인된 동작

- `채팅` 탭은 Q&A, 단어 검색, 문장 교정, FAQ, 1:1 문의 대화 목록을 보여줍니다.
- `알림` 탭은 학습 리마인더, 응원 알림, 공지사항을 보여줍니다.
- Q&A 빠른 실행 후 질문을 입력하면 답변이 채팅 영역에 추가됩니다.

## 주의점

- 일부 시험 화면에서는 AI 튜터 패널이 시험 버튼을 가릴 수 있습니다.
