# 홈 V1 IA

URL: `/home.html`

## 화면 목적

학습 현황을 한눈에 보여주고, 사용자가 새 학습이나 이어하기로 빠르게 진입하는 대시보드입니다.

## 주요 구성요소

- V1/V2 전환 버튼
- 인사 영역: 사용자 이름, 플랜, 상위 페이스, 배지, XP
- 목표 등급 카드: 목표 급수, 시험일까지 남은 날짜
- 주간 학습 요약: 총 학습 시간, 푼 문제 수, 출석 일수
- 영역별 실력: 듣기, 읽기, 쓰기 점수와 합격 예측
- 새로운 학습 카드
  - 듣기/읽기 집중
  - 쓰기 집중 연습
  - 실전 모의고사
- 이어하기 카드: 51번 실용문 빈칸 문제
- 약점 카드: 쓰기 53번 도표 분석
- 소식 및 이벤트 카드

## 사용자 조작 요소

- `V2 (쓰기 집중)`: 쓰기 중심 홈으로 이동합니다.
- `듣기/읽기 집중`: AI 맞춤 문제 생성 화면으로 이동합니다.
- `쓰기 집중 연습`: 쓰기 집중 연습 설정 화면으로 이동합니다.
- `이어하기`: 51번 쓰기 연습 화면으로 이동합니다.
- `WEAK POINT`: 53번 쓰기 연습 화면으로 이동합니다.
- `더보기 >`: 게시판으로 이동합니다.
- `실전 모의고사`: 클릭 가능한 카드입니다.

## 확인된 동작

- 듣기/읽기 카드는 문제 생성 화면으로 이동했습니다.
- 쓰기 집중 연습 카드는 쓰기 연습 설정 화면으로 이동했습니다.
- 이어하기는 51번 쓰기 화면으로 이동했습니다.
- 약점 공략은 53번 쓰기 화면으로 이동했습니다.
- 더보기는 게시판으로 이동했습니다.
- 실전 모의고사 카드는 클릭 가능했지만 탐색 시 별도 이동은 확인되지 않았습니다.
