# 단어장 IA

URL: `/my_vocabulary.html`

## 화면 목적

저장한 단어를 카드 또는 목록 형태로 복습하는 화면입니다.

## 주요 구성요소

- 총 단어 수
- 카드 보기/리스트 보기 전환
- 단어 카드
- 뜻과 예문
- 이전/다음 단어 이동
- 암기 완료 버튼
- 목록 보기 상태의 테이블
- 페이지네이션

## 사용자 조작 요소

- `리스트 보기`: 카드형 단어장을 표 형태로 바꿉니다.
- `카드 보기`: 표 형태에서 카드형으로 돌아갑니다.
- 이전/다음 단어 이동
- `의미 & 예문 보기`
- `암기 완료! 목록에서 제거`

## 확인된 동작

- `리스트 보기`를 누르면 표 형태 목록으로 전환됩니다.
- `암기 완료! 목록에서 제거`를 누르면 제거 전 확인 창이 뜹니다.

## 연결 화면

- 내 서재
- AI 튜터 단어 검색
