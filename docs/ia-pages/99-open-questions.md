# 확인 필요 및 남은 위험

확인 기준: 배포 사이트를 Playwright MCP로 직접 탐색한 결과입니다.

## 확인 필요 항목

- 홈 V1의 `실전 모의고사` 카드 클릭은 화면 이동이 확인되지 않았습니다.
- 내 서재의 드래그 기반 문제 바구니 동작은 화면 안내로 확인했지만, 실제 드래그 완료 후 풀이 시작까지는 추가 검증이 필요합니다.
- 일부 검색 입력은 값을 넣어도 즉시 목록이 줄어드는 변화가 명확히 보이지 않았습니다.
- 언어 전환은 메뉴와 주요 UI 문구 중심으로 적용되고, 일부 학습 데이터와 피드백 문구는 한국어로 유지됩니다.
- AI 튜터는 빠른 실행 문맥에 따라 사용자의 질문을 의도와 다르게 분류할 수 있습니다.

## 사용자 관점의 영향

- 사용자는 일부 버튼을 눌렀을 때 기대한 이동이나 필터링이 일어났는지 헷갈릴 수 있습니다.
- 내 서재의 핵심 행동인 문제 바구니 담기가 명확히 확인되지 않으면 복습 시작 흐름을 이해하기 어렵습니다.
- 시험 화면에서 AI 튜터가 버튼을 가리면 시험 진행 중 조작 실수로 이어질 수 있습니다.
