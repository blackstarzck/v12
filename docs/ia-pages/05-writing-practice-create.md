# 쓰기 집중 연습 설정 IA

URL: `/writing_practice_create.html`

현재 React route: `/writing/setup`

## 화면 목적

쓰기 51번부터 54번까지의 유형과 주제를 골라 쓰기 연습을 시작하는 화면입니다.

## 주요 구성요소

- 문제 유형 선택
  - 51번 실용문 빈칸
  - 52번 설명문 빈칸
  - 53번 도표/그래프
  - 54번 자신의 생각
- 주제 검색 입력창
- 주제 칩
  - 여행
  - 일상생활
  - 사회
  - 환경
  - 기술
  - 기타 주제
- 생성 버튼
- AI 맞춤형 문제 미리보기

## 사용자 조작 요소

- 문제 유형 버튼
- 주제 검색창
- 주제 칩
- 생성 버튼

## 확인된 동작

- 처음에는 생성 버튼이 비활성 상태이며 `문제 유형을 선택해주세요`라고 표시됩니다.
- 53번 유형을 선택하면 버튼 문구가 `53번 문제 생성하기`로 바뀌고 활성화됩니다.
- 주제 `환경`을 선택한 뒤 생성하면 53번 쓰기 연습 화면으로 이동했습니다.
