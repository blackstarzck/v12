# 공통 레이아웃 및 전역 요소 IA

확인 기준: 배포 사이트를 Playwright MCP로 직접 탐색한 화면입니다.

## 화면 목적

모든 주요 화면에서 사용자가 같은 방식으로 이동하고, 언어를 바꾸고, 계정 정보를 확인할 수 있게 하는 기본 틀입니다.

## 주요 구성요소

- 왼쪽 사이드바
  - `TALKPIK AI` 로고
  - `내 서재`
  - `단어장`
  - `쓰기 보관함`
  - `모의고사 결과`
  - `게시판`
- 사용자 영역
  - 사용자명: `김토픽 님`
  - 플랜: `Premium Plan`
  - 클릭 시 프로필 설정으로 이동
- 언어 버튼
  - `KO`
  - `VI`
  - `EN`
- 전역 AI 튜터 플로팅 버튼

## 사용자 조작 요소

- 사이드바 메뉴를 눌러 주요 화면으로 이동합니다.
- 사용자 영역을 눌러 프로필 설정으로 이동합니다.
- 언어 버튼을 눌러 화면 문구 언어를 바꿉니다.
- AI 튜터 버튼을 눌러 보조 채팅 패널을 엽니다.

## 확인된 동작

- `KO`, `VI`, `EN`을 누르면 메뉴와 주요 화면 문구가 바뀝니다.
- 일부 학습 데이터와 AI 피드백 문장은 언어를 바꿔도 한국어로 남아 있었습니다.

## 연결 화면

- 홈
- 내 서재
- 단어장
- 쓰기 보관함
- 모의고사 결과
- 게시판
- 프로필 설정
- AI 튜터 패널
