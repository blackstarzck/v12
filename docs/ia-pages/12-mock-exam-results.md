# 모의고사 결과 IA

URL: `/mock_exam_results.html`

## 화면 목적

최근 모의고사 성적과 영역별 실력, 성적 변화 추이, 응시 기록을 확인하는 화면입니다.

## 주요 구성요소

- 새 모의고사 응시 버튼
- TOPIK I/TOPIK II 전환
- 최근 등급
- 평균 점수
- 영역별 집중 연습 점수
- 총 응시 횟수
- 영역별 실력
- 성적 변화 추이
- 응시 기록 테이블
- 전체보기 링크

## 사용자 조작 요소

- `새 모의고사 응시`: 실전 모의고사 생성 화면으로 이동합니다.
- TOPIK I/TOPIK II 전환
- 최근 10회/전체보기
- 회차별 모의고사/영역별 읽기/영역별 듣기 차트 전환
- 성적표 버튼
- 전체보기 링크

## 연결 화면

- 전체 응시 기록
- 실전 모의고사 생성
