# 실전 모의고사 풀이 IA

URL 예: `/mock_test_exam.html?...`

## 화면 목적

제한 시간 안에 실제 시험처럼 문제를 푸는 화면입니다.

## 주요 구성요소

- 시험명: `TALKPIK TOPIK I 실전 모의고사`
- 남은 시간
- 일시정지 버튼
- 현재 영역과 진행 상태
- 시험 종료 버튼
- 문제 지문과 보기
- 오디오 재생 영역
- 이전/다음
- OMR 답안지 버튼

## 사용자 조작 요소

- 오디오 재생
- 보기 선택
- 이전/다음 문제 이동
- 답안지 열기
- 시험 종료

## 확인된 동작

- `답안지 (OMR)`를 누르면 번호별 답안 현황이 열립니다.
- AI 튜터 패널이 열려 있으면 답안지 버튼을 가릴 수 있어 닫은 뒤 조작해야 했습니다.

## 연결 화면

- 실전 모의고사 생성
- 모의고사 결과
