# AI 맞춤 문제 생성 IA

URL: `/practice_create.html`

현재 React route: `/practice/create`

## 화면 목적

읽기 또는 듣기 문제를 사용자가 조건에 맞게 생성하도록 돕는 설정 화면입니다.

## 주요 구성요소

- 학습 영역 선택
  - 읽기
  - 듣기
- 목표 시험 선택
  - TOPIK I
  - TOPIK II
- 목표 등급 선택
  - 전체
  - 3급
  - 4급
  - 5급
  - 6급
- 문제 유형 목록
- 생성할 문제 수
- AI 맞춤형 문제 미리보기

## 사용자 조작 요소

- 읽기/듣기 영역 버튼
- TOPIK I/TOPIK II 버튼
- 등급 버튼
- 문제 유형 카드
- `AI 문제 생성 시작하기`

## 확인된 동작

- 문제 유형을 선택하지 않고 생성 버튼을 누르면 `문제 유형을 선택해주세요!` 알림이 뜹니다.
- 문제 유형을 선택한 뒤 생성하면 문제 풀이 화면으로 이동합니다.

## 연결 화면

- 홈 V1
- 문제 풀이
