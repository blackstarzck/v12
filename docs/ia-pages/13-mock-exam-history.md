# 전체 응시 기록 IA

URL: `/mock_exam_history.html`

## 화면 목적

모든 모의고사 응시 기록을 표로 확인하고 필터링하는 화면입니다.

## 주요 구성요소

- 새 모의고사 응시 버튼
- TOPIK I/TOPIK II 전환
- 유형 필터
  - 전체
  - 회차별
  - 읽기
  - 듣기
  - 쓰기
- 정렬
  - 최신순
  - 과거순
  - 점수 높은순
  - 점수 낮은순
- 응시 기록 테이블
- 성적표 버튼

## 사용자 조작 요소

- 필터 버튼
- 정렬 선택
- 성적표 보기
- 새 모의고사 응시

## 연결 화면

- 모의고사 결과
- 실전 모의고사 생성
