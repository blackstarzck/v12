# 내 서재 IA

URL: `/my_library.html`

현재 React route: `/library`

## 화면 목적

저장한 문제를 관리하고, 폴더나 문제를 학습 바구니에 담아 복습하는 화면입니다.

## 주요 구성요소

- TOPIK I/TOPIK II 전환
- 빠른 학습 카드
  - 취약 문항 공략
  - 전체 무작위 복습
- `내 북마크 직접 관리하기`
- 모든 문항 및 폴더 관리
- 폴더 접기
- 목록 편집
- 필터
  - 영역
  - 유형
  - 태그
- 정렬
  - 최신 저장순
  - 오래된 순
  - 유형순
- 폴더별 개수
- 문제 목록
- 문제 바구니

## 사용자 조작 요소

- TOPIK I/TOPIK II 전환
- 빠른 학습 카드 클릭
- 필터와 정렬 선택
- 문제별 `풀기`
- 문제 또는 폴더를 문제 바구니로 드래그
- `풀기 시작`

## 확인된 동작과 주의점

- 화면 안내상 드래그해서 문제 바구니에 담는 구조입니다.
- 몇몇 카드와 `풀기` 버튼은 탐색 중 같은 화면에 머무르는 동작으로 관찰되었습니다.

## 연결 화면

- 문제 풀이
- 공통 사이드바 화면
