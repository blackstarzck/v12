# 실전 모의고사 생성 IA

URL: `/mock_test_setup.html`

## 화면 목적

실전 시험과 같은 조건의 모의고사를 시작하기 전 시험 유형과 구성을 확인하는 화면입니다.

## 주요 구성요소

- 회차별 실전 모의고사
- 영역별 집중 연습
- TOPIK I/TOPIK II 선택
- 시험 구성 안내
- 예상 소요 시간
- 총 문항 수
- 실전 모의고사 가이드

## 사용자 조작 요소

- 시험 유형 선택
- TOPIK 단계 선택
- `시험 시작하기`

## 확인된 동작

- `시험 시작하기`를 누르면 실제 시험 풀이 화면으로 이동합니다.

## 연결 화면

- 모의고사 결과
- 전체 응시 기록
- 실전 모의고사 풀이
