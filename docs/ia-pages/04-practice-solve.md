# 문제 풀이 IA

URL 예: `/practice_solve.html?...`

## 화면 목적

생성된 읽기 또는 듣기 문제를 실제로 풀고 정답을 확인하는 화면입니다.

## 주요 구성요소

- 상단 진행 정보: 예를 들어 `READING | 1 / 3`
- 이전/다음 이동 버튼
- 북마크 버튼
- 지문 영역
- 문제 문항
- 보기
- `정답 확인` 버튼

## 사용자 조작 요소

- 이전/다음 버튼으로 문제 이동
- 보기 선택
- 정답 확인
- 북마크 저장
- 뒤로가기

## 확인된 동작

- 문제 생성 화면에서 유형을 선택한 뒤 생성하면 이 화면으로 이동했습니다.
- 읽기 예시 화면에서는 지문, 문제, 보기, 정답 확인 버튼이 노출됐습니다.

## 연결 화면

- AI 맞춤 문제 생성
- 내 서재
