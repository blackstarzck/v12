# 게시판 IA

URL: `/board.html`

현재 React route: `/board`

## 화면 목적

공지사항과 이벤트를 확인하는 화면입니다.

## 주요 구성요소

- 탭
  - 공지사항
  - 이벤트
- 공지 분류
  - 전체
  - 중요
  - 안내
  - 학습
  - 점검
  - 당첨
- 검색창
- 공지사항 테이블
- 이벤트 카드 목록
- 페이지네이션

## 사용자 조작 요소

- 공지사항/이벤트 탭 전환
- 분류 필터
- 제목 검색
- 페이지 이동
- 게시글 제목 클릭

## 확인된 동작

- 이벤트 탭은 진행중/종료 이벤트 카드 목록을 보여줍니다.
- 중요 필터를 누르면 중요 공지만 1건으로 줄어듭니다.
- 공지 제목을 누르면 공지 상세 화면으로 이동합니다.

## 연결 화면

- 공지 상세
- 홈 소식 영역
