# 쓰기 보관함 IA

URL: `/writing_feedback_list.html`

현재 React route: `/writing/feedback`

## 화면 목적

작성한 쓰기 답안의 AI 피드백과 임시 저장 답안을 모아보는 화면입니다.

## 주요 구성요소

- 탭
  - 완료된 피드백
  - 임시 보관 내역
- 검색창
- 필터 버튼
- 정렬 선택
- 목록 편집
- 피드백 카드 목록
- 페이지네이션

## 사용자 조작 요소

- `완료된 피드백`: 점수와 피드백 완료 목록을 보여줍니다.
- `임시 보관 내역`: 작성 중인 답안을 보여줍니다.
- 검색창과 검색 실행 버튼
- `필터`: 문제 유형과 상태 필터를 엽니다.
- 정렬 선택
- 목록 편집
- 피드백 카드 클릭

## 확인된 동작

- 임시 보관 탭에는 작성 중 상태, 자동 저장 여부, 작성 글자 수가 보입니다.
- 필터 버튼을 누르면 51/52/53/54번, New, 다시 푼 문제 조건을 선택하는 필터 영역이 열립니다.

## 연결 화면

- 쓰기 피드백 상세
- 쓰기 연습 51번
- 쓰기 연습 53번
