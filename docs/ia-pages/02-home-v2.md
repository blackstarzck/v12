# 홈 V2 IA

URL: `/home_v2.html`

## 화면 목적

쓰기 학습에 집중한 축약형 홈입니다. 전체 학습보다 쓰기 연습과 쓰기 보관함 접근을 앞세웁니다.

## 주요 구성요소

- V1/V2 전환
- 간소화된 사이드 메뉴
  - 쓰기 연습
  - 쓰기 보관함
  - 게시판
  - 단어장
  - 내 서재
- 목표 등급, 배지, XP
- 주간 학습 요약
- 핵심 학습 카드: 쓰기 집중 연습
- 이어하기 카드: 51번 실용문
- AI 추천 약점 공략: 쓰기 53번
- 소식 및 이벤트

## 사용자 조작 요소

- `V1 전체`: 기본 홈으로 돌아갑니다.
- `쓰기 연습`: 쓰기 집중 연습 설정 화면으로 이동합니다.
- `쓰기 보관함`: 쓰기 보관함으로 이동합니다.
- `게시판`: 게시판으로 이동합니다.
- `단어장`: 단어장으로 이동합니다.
- `내 서재`: 내 서재로 이동합니다.
- 핵심 학습 카드와 약점 공략 카드로 쓰기 학습을 시작합니다.
